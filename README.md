## 🧠 About Me

I'm **Salman** — a software engineer with professional experience in Java and hands-on internship experience building AI systems. I like building things that actually run: automation pipelines, self-hosted infrastructure, and AI workflows that don't depend on someone else's cloud.

- ☕ Java engineer by trade, AI tinkerer by conviction
- 🤖 Built self-hosted AI stacks with **Ollama, n8n, and Docker** during my Erasmus+ internship in Cyprus
- 🎓 Currently doing my **Masters in CS @ BTU Cottbus-Senftenberg**
- 🔥 Obsessed with **local-first AI** — if it needs a subscription, I'll self-host it
- 📸 Find me on **Instagram** when I'm not staring at a terminal
